    Extensiones Peligrosas

      Explicación de los colores

      Peligroso

      Este se refiere a permisos con los cuales la extensión puede ser vulnerable a un ataque si no se tiene precaución por parte del programador.

      Acceso exesivo

      Esta categoria se refiere a permisos con los cuales la extensión tiene permiso de manipular el navegador, la pagina web actual o las peticiones web en curso sin tener que pedirle permiso o informar al usuario en el momento. Tambien se refiere a permisos que conceden acceso a datos muy sensibles

      Acceso moderado

      Esta categoria se refiere a permisos con los cuales la extensión tiene acceso a datos que se podrian considerar sensibles o privados.

      Sin peligro

      Esta categoria se refiere a permisos que dentro de mi analisis e investigacion, no encontre algun peligro inminente, esto no significa que no pueda haber, solo que no encontre.
